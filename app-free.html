To connect Gmail and Google Calendar, you need a free Google Cloud OAuth Client ID. This takes ~5 minutes to set up once.

1
Go to console.cloud.google.com → Create a New Project (name it anything, e.g. "Job Dashboard")
2
In the left menu go to APIs & Services → Library and enable both Gmail API and Google Calendar API
3
Go to APIs & Services → OAuth consent screen → choose External → fill in App name and your email → Save
4
Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID → Application type: Web application
5
Under Authorized JavaScript origins, add the URL where you're opening this file (e.g. http://localhost or your hosted URL)
6
Click Create → copy the Client ID (looks like 123456-abc.apps.googleusercontent.com) and paste it below
⚠️ If you see "App not verified" when signing in, click Advanced → Go to [App Name] (unsafe) — this is expected for personal apps not published to the public.

To search LinkedIn, Indeed, and Glassdoor listings directly in the dashboard, enter a free RapidAPI JSearch key.

1
Go to rapidapi.com/letscrape-6bRBa3QguO5/api/jsearch
2
Sign up for a free account → subscribe to the JSearch API (free tier: 200 requests/month)
3
Copy your X-RapidAPI-Key from the code examples panel and paste below
✓ You can skip this and still manually add jobs to your pipeline. You can always come back and add the key in Settings.
✅

You're all set!

Your credentials are saved locally in your browser. Nothing is sent to any server.
